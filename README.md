# SCMDB 繁體中文語言包（台灣玩家風格）

這是給 [SCMDB](https://scmdb.net/) 使用的繁體中文語言包。

## 使用方式

將下列網址貼到 SCMDB 的語言包欄位：

```text
https://raw.githubusercontent.com/a27646415/scmdb-zh-tw/main/lang-ZH-TW.json
```

或直接用：

```text
https://scmdb.net/?lang=https://raw.githubusercontent.com/a27646415/scmdb-zh-tw/main/lang-ZH-TW.json
```

## 翻譯原則

| 類型 | 原則 |
|---|---|
| 船艦名稱 | 保持英文 |
| 製造商 | 保持英文 |
| 陣營 | 保持英文 |
| 地點 | 保持英文 |
| 礦物 | 繁中 + 英文並列，例如：鈦礦（Titanium Ore） |
| 貨物 | 繁中 + 英文並列，例如：醫療物資（Medical Supplies） |
| 任務 | 翻成自然繁體中文 |
| 介面 | 翻成繁體中文 |

## 更新方式

1. 下載最新 SCMDB 語言包或其他語言包。
2. 依照本專案格式更新 `lang-ZH-TW.json`。
3. 上傳到 GitHub main 分支。
4. SCMDB 會讀取 Raw GitHub URL。
